## 🔐 Objective
Design and secure a small enterprise network using VLANs, ACLs, DHCP, NAT, and Router-on-a-Stick.

## 🧩 Configuration Components
- VLANs: 10 (Management), 20 (Finance), 30 (Guest)
- Trunking configuration between switches and router
- Inter-VLAN routing (Router-on-a-Stick)
- ACL to block Guest → Finance
- DHCP for each VLAN
- NAT for internet access
- Basic switch & router hardening
